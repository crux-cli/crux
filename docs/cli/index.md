# CLI Reference

Complete reference for all `crux` commands.

## Setup

| Command | Description |
|---------|-------------|
| [`crux setup`](setup.md) | Initialize `~/.crux/` and environment |
| [`crux doctor`](doctor.md) | Diagnose and auto-fix environment issues |

## Registry

| Command | Description |
|---------|-------------|
| [`crux add`](add.md) | Register a new MCP or skill |
| [`crux remove`](remove.md) | Unregister an MCP or skill |
| [`crux list`](list.md) | List everything in the registry |
| [`crux search`](search.md) | Search the official MCP Registry |
| [`crux upgrade`](upgrade.md) | Update cloned sources to latest |

## Project

| Command | Description |
|---------|-------------|
| [`crux init`](init.md) | Create a project with `crux.json` |
| [`crux install`](install.md) | Add MCPs/skills to project and sync |
| [`crux uninstall`](uninstall.md) | Remove MCPs/skills from project and sync |
| [`crux sync`](sync.md) | Generate `.mcp.json` from `crux.json` |
| [`crux status`](status.md) | Show MCP server health |

## Secrets

| Command | Description |
|---------|-------------|
| [`crux secret`](secret.md) | Manage MCP secrets (set, get, list, delete) |

## Sandbox

| Command | Description |
|---------|-------------|
| [`crux run`](run.md) | Run an agent in an isolated sandbox |

## Version

| Command | Description |
|---------|-------------|
| [`crux version`](version.md) | Show crux-cli version |
