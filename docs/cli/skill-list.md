# crux skill list

List all registered skills.

## Usage

```bash
crux skill list [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |

## Description

Displays all skills registered in `~/.crux/skills.json`, including their source type (github, local) and tags.

## Examples

```bash
# List in table format
crux skill list

# Output as JSON
crux skill list --json
```
